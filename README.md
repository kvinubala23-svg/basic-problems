# basic-problems

## 📌 Overview
This repository contains a set of **basic Java programs** designed to practice fundamental concepts such as arrays, loops, input handling, conditionals, and simple algorithms.  
Each file demonstrates a specific problem or logic implementation with a `main` method for execution.

---

## 📂 Programs Included

- **Array.java** → Defines an `Array` class with a `main` method to output numbers.
- **Basic.java** → Simple Java program demonstrating basic syntax.
- **Car.java** → Contains the `Basic` class and a `main` method for car-related logic.
- **Change.java** → Handles user input and performs calculations.
- **Fact.java** → Implements a factorial method and runs it in the `main` method.
- **Interest.java** → Calculates simple interest using an `Interest` class.
- **Main.java** → Demonstrates input handling and loop functionality.
- **OddNumber.java** → Prints odd numbers from 1 to 100.
- **PalindromeCheck.java** → Checks whether a given string/number is a palindrome.
- **Solution.java** → Implements `minimumCost` method to calculate total cost.
- **SumofNum.java** → Calculates the sum of odd numbers from 1 to 100.
- **TotalNumOfOddNum.java** → Counts the total number of odd numbers from 1 to 100.
